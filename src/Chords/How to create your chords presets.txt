How to Create Your Own Custom Chord Preset

You can create your own custom chord preset by creating a new .ini file like "My-Custom-Chords.ini" in the Chords folder. Follow these instructions to set up your custom chords:

1. Open a text editor (like Notepad).
2. For each chord, create a new [Shortcut_X] section, where X is a number from 1 to 48.
3. Under each section, specify the ChordName, ChordInterval, and ShortcutKey.

Here's the format for each chord:

[Shortcut_X]
ChordName=YOUR_CHORD_NAME
ChordInterval=INTERVALS
ShortcutKey=YOUR_SHORTCUT

Examples:

1. Using function keys (F1-F12):

[Shortcut_1]
ChordName=MAJOR
ChordInterval=0 4 7
ShortcutKey=F1

2. Using Alt modifier (represented by !):

[Shortcut_2]
ChordName=MINOR
ChordInterval=0 3 7
ShortcutKey=!F1

3. Using Ctrl modifier (represented by ^):

[Shortcut_3]
ChordName=DOMINANT 7th
ChordInterval=0 4 7 10
ShortcutKey=^F1

4. Using Shift modifier (represented by +):

[Shortcut_4]
ChordName=AUGMENTED
ChordInterval=0 4 8
ShortcutKey=+F1

Notes:
- You can use any key on your keyboard but F1 through F12 are recommended.
    - List of key names https://www.autohotkey.com/docs/v2/KeyList.htm
- Modifiers follow AutoHotkey conventions:
    - ! for Alt
    - ^ for Ctrl
    - + for Shift.
- ChordInterval represents the semitones from the root note. For example, 0 4 7 represents a major chord (root, major third, perfect fifth). Numbers must be separated by a single space " ".
- Negative semitones are accepted in the ChordInterval field. Example: -12 0 12 (this repeat the selected note an octave down and an octave up)

5. Save your file as "My-Custom-Chords.ini" in the Chords folder.
6. Restart the application using the _Reload_ from the tray menu, then select your new custom preset from the dropdown menu.

Remember, you can create up to 48 chord shortcuts in a single preset file. Enjoy creating your custom chord palette!
